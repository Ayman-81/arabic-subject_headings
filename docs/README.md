# توثيق مشروع رؤوس الموضوعات العربية

## نظرة عامة

هذا المجلد يحتوي على وثائق المشروع التقنية ودليل المستخدم.

## المحتويات

- `API.md` - توثيق نقاط API
- `ARCHITECTURE.md` - بنية النظام
- `CONTRIBUTING.md` - دليل المساهمة

## نقاط API المتاحة

| المسار | الطريقة | الوصف |
|---|---|---|
| `/api/search?q=كلمة` | GET | البحث في رؤوس الموضوعات |
| `/api/categories` | GET | جلب جميع التصنيفات |
| `/api/sources` | GET | جلب جميع المصادر |
| `/api/subjects/<id>` | GET | جلب رأس موضوع بالمعرف |
| `/api/stats` | GET | إحصائيات النظام |

## متطلبات التشغيل

```bash
pip install -r requirements.txt
python app.py
```
